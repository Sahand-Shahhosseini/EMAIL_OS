# 02 Claim Status

Date: 2026-03-27
Status: Bootstrap scaffold
Purpose: record the minimum status semantics used inside `EMAIL_OS`.

## Minimum status vocabulary
- draft
- active
- triaged
- pending-action
- blocked
- archived
- failed-sync
- deprecated

## Hard rule
A status inside Email-OS describes operational email state, not general account truth or project completion truth.
